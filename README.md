# 스타 세이비어 이벤트 인식기
> [스타세이비어 아르카나 DB 사이트](https://kanohban.github.io/StarSaviorArcanaDB/index.html) 를 기반으로 제작하였습니다.

- 현재 21:9 모니터에서 테스트 진행되었으나, 16:9 화면에서는 테스트가 진행되지 않았습니다.
- 간혈적으로 이벤트 인식이 안되는 오류가 있습니다.
- 비동기를 사용하여 UI Thread가 멈추진 않으나, 아르카나 인식부로 넘어가게 되면, CPU를 많이 사용하는 문제가 있습니다. (이미지 인식으로 아르카나 종류 판단)
  - 해당 부분은 로직을 고민중입니다.
